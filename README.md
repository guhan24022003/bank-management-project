# Bank Management System
A professional banking application built with Python that implements core financial operations and secure account management.

# Key Features
#Authentication:
- Sign In / Sign Up: Check if a user is registered or create a new account.
- User Registration: Securely onboard new users into the system.

#Account Management:
- Credit and Withdraw: Add or remove funds from the account easily.
- Funds Transfer: Securely send money to another account.
- Account Details: View and update personal information.
- Balance Enquiry: Check real-time available balance.

#Banking Facilities:
- Transaction History: Access a detailed log of all account activities.
- Date and Time Tracking: Accurate timestamps for every transaction.
- OOPs Implementation: Developed using Object-Oriented Programming for modular and clean code.

# Tech Stack
- Language: Python
- Programming Paradigm: Object-Oriented Programming (OOPs)
